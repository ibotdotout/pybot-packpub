# Pybot-Packpub

Automatic get free ebook from packpub to your account

## How to Use

```sh
# Edit {USER} and {PASSWORD} in getebook.txt first !!!
$ pybot
```

or

```sh
$ pybot --variable USER:<user> --variable PASSWORD:<password> getbook.txt
```

## Requiremnet
- python
- robotframework
- robotframework-selenium
- phatomjs

or

- docker
